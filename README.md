# WeyDocs — Landing Site

Marketing and onboarding site for the WeyDocs mobile app. Built with Next.js, TypeScript, and Tailwind CSS.

WeyDocs is a document tracking app that helps you never miss a renewal deadline — iqama, visa, insurance, passport, and more.

## Pages

- `/` — Landing page with hero section and developer note
- `/waitlist` — Early access signup (Tally form embed)
- `/contact` — Contact form (Tally form embed)
- `/privacy` — Privacy Policy
- `/terms` — Terms of Service

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Font:** Inter
- **Forms:** Tally embeds
